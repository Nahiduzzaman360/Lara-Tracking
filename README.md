# Lara Tracking (Realtime user location tracking application using Vue.js and Laravel)

## Installation

Download or Pull the repositories, go to the project directory, then run those commands:

```bash
  composer install
``` 
```bash
  npm install
``` 
```bash
  rename .env.example to .env
```
```bash
  php artisan migrate
```
```bash
  npm run dev
``` 
```bash
  php artisan serve
```
